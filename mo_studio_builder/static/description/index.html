MO Studio Builder

Visual App Builder and Studio-like Editor for Odoo 17

Create custom Odoo applications, add fields, edit page layouts, configure properties, and update generated views from a visual editor.

Created by abdulrahman sammour

Header Studio Button

Open the builder from the Odoo top bar while working inside supported pages.

Live Preview Editor

Preview fields and layout changes before applying them to the real Odoo view.

Field Properties

Edit labels, help text, readonly, required, relations, and selection options.

What You Can Create

  • Custom approval, request, tracking, rental, support, and operations apps.
  • Form and list views with text, numeric, date, boolean, image, relation, monetary, and selection fields.
  • Generated Odoo menus, view shortcuts, access shortcuts, reports, and automation links.

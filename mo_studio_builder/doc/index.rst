MO Studio Builder
=================

MO Studio Builder is a visual app builder for Odoo 17 created by
abdulrahman sammour.

Main Features
-------------

* Create custom Odoo applications from the user interface.
* Add generated apps to the normal Odoo apps menu with their own icon.
* Open the Studio editor from the Odoo header while working on supported pages.
* Add text, multiline, HTML, integer, decimal, monetary, date, datetime,
  checkbox, selection, many2one, many2many, file, and image fields.
* Configure labels, technical names, help text, required and readonly options.
* Configure selection options and relation models from the properties panel.
* Preview form layout changes before applying them to the real Odoo views.
* Update the generated app after the design is approved.
* Open shortcuts for fields, views, access rules, reports, and automations.

Recommended Use
---------------

Use MO Studio Builder for internal business tools such as approvals, requests,
operations trackers, rental records, support flows, and custom administrative
applications.

Compatibility
-------------

This module is built for Odoo 17 Community and Enterprise.
